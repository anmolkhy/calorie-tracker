## [1.0.0] — 2026-05-04

Initial release. Core tracking loop fully functional.

### Added
- Daily food logging — search food database, enter grams, log instantly
- Fixed meal templates — create reusable meals, log everything in one tap
- Live macro calculation — calories, protein, carbs, fat vs. daily goals
- 7-day history — bar chart and daily breakdown
- Curated food database — ~60 foods with macros verified from product labels (Atom Whey, ID Fresh Paneer, Licious Chicken Curry Cut, Quaker Oats, Superyou Wafers) and USDA/IFCT data
- Custom food entry — add any food with your own label data
- Daily goals — set calorie and macro targets with visual macro split donut
- Multi-user — isolated accounts, each user's data is private
- PWA — installable on phone home screen, fullscreen native experience
- Deployed on Vercel with Turso database (Mumbai region)
